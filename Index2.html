How much JavaScript do you need to know to use Node.js?

As a beginner, it's hard to get to a point where you are confident enough in your programming abilities. While learning to code, you might also be confused at where does JavaScript end, and where Node.js begins, and vice versa.

What is recommended to learn before diving deep with Node.js?

  • Lexical Structure
  • Expressions
  • Data Types
  • Classes
  • Variables
  • Functions
  • Operator
  • Arrow Functions
  • Loops
  • Scopes
  • Arrays
  • Template Literals
  • Strict Mode
  • ES6 and beyond
  • Asynchronous JavaScript

With those concepts in mind, you are well on your road to becoming a proficient JavaScript developer.

Asynchronous Programming

The following concepts are also key to understand asynchronous programming:

  • Asynchronous programming and callbacks
  • Promises
  • Async and Await
  • Closure
  • The Event Loop
